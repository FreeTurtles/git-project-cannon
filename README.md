# Roblox Project Cannon

Roblox 게임 개발을 위한 기본 템플릿입니다.

## Stack

- Roblox-TS
- Flamework
- Rojo
- Wally (optional)

## 시작하기

1. `npm install`
2. `npm run build`
3. `npm run rojo:serve`
4. Roblox Studio에서 Rojo 플러그인으로 연결

## 구조

- `src/client`
- `src/server`
- `src/shared`
- `docs` (설계/규칙/로그)
  - `docs/project_Structure.md` (아키텍처/결정)
  - `docs/Daily_Process.md` (일일 작업일지)
- `resource` (기존 참고 코드/레퍼런스)
